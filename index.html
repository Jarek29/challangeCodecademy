Tag Name Description
<table> Table The <table> element creates a table.
<thead> Table Head Table headings clarify the meaning of data. Headings are added with the <thead> element.
<tbody> The Body The set of rows containing actual table data.
<tr> Table Row The <tr> element adds rows to a table.
<td> Table Data To add data to a row, you can use the <td> element.
<th> Table Headings Table headings clarify the meaning of data. Headings are added with the <th> element.
<colspan> Colspan Table data can span columns using the <colspan>attribute.
